## ğŸ” WHAT'S THE DIFFERENCE?

### AAB (App Bundle) Files
- **For:** Google Play Store only
- **Size:** Optimized (~50 MB)
- **Installation:** Users download from Play Store
- **Distribution:** Managed by Google
- **Best for:** Official app store release

### APK Files
- **For:** Direct installation on Android devices
- **Size:** Slightly larger (~55 MB)
- **Installation:** Manual download and install
- **Distribution:** You manage (email, website, USB, etc.)
- **Best for:** Testing, beta program, pilot users, internal distribution

---

## âœ… WHY BOTH?

**AAB for Play Store:**
- Official distribution channel
- Automatic updates
- Wider reach
- Google's security vetting

**APK for Direct Install:**
- Immediate testing (no Play Store review wait)
- Beta testers can install before Play Store release
- Pilot program participants
- Internal team testing
- Offline distribution (USB transfer)

## ğŸ“± HOW TO INSTALL APK FILES

### Step 1: Download APK
1. Click the APK download link when build completes
2. Save file to your computer
3. Transfer to Android phone (USB, email, cloud)

### Step 2: Enable Unknown Sources
1. Open **Settings** on Android device
2. Go to **Security** or **Apps**
3. Enable **"Install from Unknown Sources"**
4. Or allow installation for specific app (Chrome, Files, etc.)

### Step 3: Install
1. Open the APK file on Android device
2. Tap **"Install"**
3. Wait for installation
4. Tap **"Open"** to launch app
